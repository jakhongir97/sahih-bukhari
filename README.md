# Sahih al-Bukhari — landing page

The marketing, support, and privacy page for the **Sahih al-Bukhari** iOS app.
It doubles as the App Store **Support URL** and **Privacy Policy URL**.

Pure static site — HTML + images, no build step. Deployed with GitHub Pages and
intended to live at **https://joxon.uz**.

```
index.html            the page (self-contained CSS, no dependencies)
shots/                app screenshots
icon.png / favicon    app icon in a few sizes
```

The page is marketing only — screenshots + description; it does not host the
hadith text itself.
